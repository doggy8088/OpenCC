# 不同簡繁詞庫的特性

這個範例比較同一段簡體中文轉到不同目標詞庫時的輸出差異。

## 詞庫重點

| 目標 | 特性 |
| --- | --- |
| `t` | OpenCC 標準繁體，只做字形層級，不套用地區詞彙。 |
| `tw` | 臺灣繁體字形與部分臺灣用字。 |
| `tw2` | 臺灣常用詞彙，常見例子如「计算机」轉「電腦」。 |
| `twp` | 臺灣詞彙加慣用語，通常比 `tw2` 更口語或在地化。 |
| `hk` | 香港繁體字形與香港常用詞。 |
| `jp` | 日本新字體相關轉換。 |

## 執行

```bash
dotnet run --project examples/locale-differences
```
