# 不套用詞彙轉換

這個範例示範只做簡繁字形轉換，不套用臺灣或香港地區詞彙。

## 什麼時候使用

- 你只想把「汉语」轉成「漢語」，但不想把「软件」轉成「軟體」。
- 你需要保留原文用詞，只修正簡繁字形。
- 你正在處理專有名詞、法規文字或已經有人工詞彙規範的內容。

## 做法

使用 `OpenCC.Converter("cn", "t")`。其中 `t` 是 OpenCC 的標準繁體中繼格式，只會套用簡體到繁體字形，不會再套用 `tw`、`tw2`、`hk` 等地區詞彙。

## 執行

```bash
dotnet run --project examples/no-phrase-conversion
```
